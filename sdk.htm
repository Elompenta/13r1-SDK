The innovaphone App SDK

Welcome to the innovaphone App SDK. With this SDK you can create greate applications for the innovaphone PBX platform. Also documentation for all innovaphone protocols and APIs is included.

So this SDK helps you to create Apps running on the innovaphone App Platform as well as any other applications interfacing in some way with the innovaphone PBX.

Table of contents

PBX Overview about the innovaphone PBX system. The components and protocols used by the different components
App Platform Introduction to the innovaphone App Platform. Details about the inner components of the App Platform and how they interact.
Platform SDK Getting started setting up the Visual Studio development enviroment. How to use it to to build, deploy and debug Apps running on the innovaphone App
App Platform Interfaces and Libraries Documentation to all code provided with the App Platform SDK. This includes the libries used to provide abstract access to the operating system services, some standalone library containing useful functions, Javascript APIs on the myApps client and Javacript libraries which may be used on the App on the client itself.
