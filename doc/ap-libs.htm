The App Platform libraries and interfaces

Part of the SDK are various libriries and services

Table of content

interfaces

The sdk/common/interface folder contains all the header files for interface definitions provided by operation environment.

ilib

The sdk/common/ilib folder contains the header files for ilib library. This is a very portable libray for some fundamental features, like string manipulation, json encoding/decoding, binary trees and much more. It is a library also used within the innovaphone firmware.

lib

The sdk/common/lib folder contains the header files for more App Platform specific libraries, for functions like the websocket communications, the database access, http clients and much more.
